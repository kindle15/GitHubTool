## 🎯 Overview

### What is GithubTool?

GithubTool is a comprehensive desktop application for managing GitHub repositories. Built with Python and Tkinter, it provides an intuitive GUI for repository operations that would typically require command-line tools or multiple web browser tabs.

### Key Features

- **🔒 Safety-First Design** - Prevents accidental data loss with pre-publish checks
- **⚡ Creation Lab** - Create local repos with README/LICENSE templates and publish to GitHub
- **📊 Smart Public Browsing** - Browse public repos without authentication
- **⭐ Star Management** - Star/unstar repositories directly from the UI
- **📤 Export Tools** - Export repo lists to CSV/TXT with formatting
- **📥 Download Tools** - Download repos, releases, and assets
- **🔍 Search & Filter** - Find repos quickly with advanced filtering
- **📐 Space-Efficient Layout** - Optimized UI with resizable diagnostics panel

### Technology Stack

- **Language:** Python 3.7+
- **GUI Framework:** Tkinter (ttk widgets)
- **HTTP Library:** urllib (built-in)
- **Version Control:** Git (external dependency)
- **Configuration:** JSON file storage

### Prerequisites

```bash
# Required
Python 3.7+

GitHub Personal Access Token (for authenticated features)
Installation Steps
1. Clone Repository
bash
git clone https://github.com/kindle15/githubtool.git
cd githubtool
2. Verify Dependencies
bash
python --version   # Should be 3.7+
git --version      # Should be 2.0+
python -m tkinter  # Should open a test window
3. Configure Token (Optional)
Option A: USB / Removable Drive (Recommended, Windows)

Create a file named PersonalAccessToken.txt on any drive D: through Z:.
Paste your GitHub token as the only content (no extra spaces or newlines).
GithubTool scans drives D:–Z: automatically on startup.

Example:
  D:\PersonalAccessToken.txt
  E:\PersonalAccessToken.txt

> Note: The filename must be exactly "PersonalAccessToken.txt". Do NOT use
> ".github_token" or any other name — those will not be detected.

Option B: Environment Variable

Windows (Command Prompt):
  set GITHUB_TOKEN=ghp_your_token_here

Windows (System Properties → Environment Variables):
  Variable name:  GITHUB_TOKEN
  Variable value: ghp_your_token_here

Linux/macOS:
  export GITHUB_TOKEN="ghp_your_token_here"

There is NO home-directory file option — remove all mentions of ~/.github_token.
4. Run Application
bash
python gui.py

🔌 API Reference
GitHub API Functions
check_if_repo_exists(owner: str, repo_name: str, token: str)
Added in: v2.0.0

Check if a repository exists on GitHub.

Parameters:

owner (str): Repository owner (username)
repo_name (str): Repository name
token (str): GitHub API token
Returns: Tuple[bool, Optional[Dict]]

(True, repo_data) if repo exists
(False, None) if repo doesn't exist
Raises:

urllib.error.HTTPError: If API call fails (other than 404)
API Endpoint: GET https://api.github.com/repos/{owner}/{repo_name}

Example:

Python
exists, repo_data = check_if_repo_exists("kindle15", "my-repo", token)
if exists:
    print(f"Repo exists! Last updated: {repo_data['updated_at']}")
else:
    print("Repo doesn't exist - safe to create!")
get_authenticated_user(token: str)
Added in: v2.0.0

Get the authenticated user's username.

Parameters:

token (str): GitHub API token
Returns: str - Username of authenticated user

Raises:

urllib.error.HTTPError: If API call fails
API Endpoint: GET https://api.github.com/user

Example:

create_github_repository(name: str, description: str, private: bool, token: str)
Added in: v1.6.0

Create a new repository on GitHub.

Parameters:

name (str): Repository name
description (str): Repository description
private (bool): Whether repo should be private
token (str): GitHub API token
Returns: Dict - Repository data from GitHub API

Raises:

urllib.error.HTTPError: If creation fails (e.g., name already exists)
API Endpoint: POST https://api.github.com/user/repos

check_if_starred(owner: str, repo: str, token: str)
Added in: v1.3.0

Check if a repository is starred by the authenticated user.

Parameters:

owner (str): Repository owner
repo (str): Repository name
token (str): GitHub API token
Returns: bool - True if starred, False otherwise

API Endpoint: GET https://api.github.com/user/starred/{owner}/{repo}

HTTP Status:

204: Repo is starred
404: Repo is not starred
Example:

Python
if check_if_starred("octocat", "Hello-World", token):
    print("⭐ You've starred this repo!")
else:
    print("☆ Not starred yet")

## 🛠️ Contributing Guidelines

### Code Style

#### Python Style Guide

Follow **PEP 8** with these specifics:

**Indentation:**
```python
# Use 4 spaces (not tabs)
def my_function():
    if condition:
        do_something()
Naming Conventions:

Python
# Classes: PascalCase
class ConfigManager:
    pass

# Functions/Methods: snake_case
def get_token() -> Tuple[str, str]:
    pass

# Constants: UPPER_SNAKE_CASE
DEFAULT_CONFIG = {}

# Private: leading underscore
def _internal_helper():
    pass
Type Hints:

Python
# Use type hints for all functions
def fetch_repos(token: str) -> List[Dict]:
    pass

# Use Optional for nullable types
def get_repo(name: str) -> Optional[Dict]:
    pass

# Use Tuple for multiple returns
def check_status() -> Tuple[bool, str]:
    pass
Docstrings:

Python
def my_function(param1: str, param2: int) -> bool:
    """
    Brief description of function.
    
    Detailed description if needed.
    Can span multiple lines.
    
    Args:
        param1: Description of param1
        param2: Description of param2
    
    Returns:
        Description of return value
    
    Raises:
        ValueError: When validation fails
        
    Example:
        >>> my_function("test", 42)
        True
    """
    pass

import sqlite3
from datetime import datetime, timedelta

class RepoCache:
    """Cache repository data in SQLite."""
    
    def __init__(self, db_path: str = "repo_cache.db"):
        self.db_path = db_path
        self._init_db()
    
    def _init_db(self):
        """Initialize database schema."""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        cursor.execute("""
            CREATE TABLE IF NOT EXISTS repos (
                id INTEGER PRIMARY KEY,
                full_name TEXT UNIQUE,
                name TEXT,
                description TEXT,
                private INTEGER,
                fork INTEGER,
                stars INTEGER,
                language TEXT,
                html_url TEXT,
                cached_at TEXT
            )
        """)
        
        conn.commit()
        conn.close()
    
    def cache_repos(self, repos: List[Dict]):
        """Cache repository list."""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        now = datetime.now().isoformat()
        
        for repo in repos:
            cursor.execute("""
                INSERT OR REPLACE INTO repos 
                (id, full_name, name, description, private, fork, stars, language, html_url, cached_at)
                VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
            """, (
                repo.get("id"),
                repo.get("full_name"),
                repo.get("name"),
                repo.get("description"),
                int(repo.get("private", False)),
                int(repo.get("fork", False)),
                repo.get("stargazers_count", 0),
                repo.get("language"),
                repo.get("html_url"),
                now
            ))
        
        conn.commit()
        conn.close()
    
    def get_cached_repos(self, max_age_hours: int = 24) -> List[Dict]:
        """Get cached repositories (if not too old)."""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        cutoff = (datetime.now() - timedelta(hours=max_age_hours)).isoformat()
        
        cursor.execute("""
            SELECT id, full_name, name, description, private, fork, stars, language, html_url
            FROM repos
            WHERE cached_at > ?
        """, (cutoff,))
        
        repos = []
        for row in cursor.fetchall():
            repos.append({
                "id": row[0],
                "full_name": row[1],
                "name": row[2],
                "description": row[3],
                "private": bool(row[4]),
                "fork": bool(row[5]),
                "stargazers_count": row[6],
                "language": row[7],
                "html_url": row[8]
            })
        
        conn.close()
        return repos

class CachedGitHubAPI:
    """GitHub API client with caching."""
    
    def __init__(self, token: str, cache_duration: int = 300):
        self.token = token
        self.cache_duration = cache_duration  # seconds
        self.cache: Dict[str, Tuple[Any, float]] = {}
    
    def get(self, url: str) -> Any:
        """Get data with caching."""
        now = time.time()
        
        # Check cache
        if url in self.cache:
            data, timestamp = self.cache[url]
            if now - timestamp < self.cache_duration:
                return data
        
        # Fetch fresh data
        data, _ = _http_get_json(url, self.token)
        self.cache[url] = (data, now)
        
        return data
    
    def invalidate(self, url: str = None):
        """Invalidate cache."""
        if url:
            self.cache.pop(url, None)
        else:
            self.cache.clear()

    """Actual long-running operation."""

Pagination Optimization
Problem: Fetching all pages is slow

Solution: Lazy loading

Python
def fetch_repos_lazy(self, token: str, callback: Callable[[List[Dict]], None]):
    """Fetch repos with lazy loading."""
    
    def fetch_thread():
        page = 1
        while True:
            url = f"https://api.github.com/user/repos?per_page=100&page={page}"
            data, hdrs = _http_get_json(url, token)
            
            if not isinstance(data, list) or not data:
                break
            
            # Update UI immediately with this page
            self.after(0, lambda d=data: callback(d))
            
            link = hdrs.get("Link", "")
            if 'rel="next"' not in link:
                break
            
            page += 1
    
    threading.Thread(target=fetch_thread, daemon=True).start()

def validate_repo_name(name: str) -> bool:
    """Validate repository name."""
    if not name:
        return False
    
    # GitHub rules:
    # - Max 100 characters
    # - Alphanumeric, hyphen, underscore, period
    # - Can't start with period or hyphen
    
    if len(name) > 100:
        return False
    
    if name[0] in ('.', '-'):
        return False
    
    import re
    if not re.match(r'^[a-zA-Z0-9._-]+$', name):
        return False
    
    return True

# Usage:
repo_name = self.name_entry.get().strip()

                        "Repository name must be:\n"
                        "- 1-100 characters\n"
                        "- Alphanumeric, hyphen, underscore, period\n"
                        "- Can't start with period or hyphen")
    return
Path Sanitization
Prevent path traversal attacks:

def sanitize_path(base_dir: Path, user_path: str) -> Path:
    """Sanitize user-provided path."""
    # Resolve to absolute path
    full_path = (base_dir / user_path).resolve()
    
    # Ensure it's within base_dir
    try:
        full_path.relative_to(base_dir)
        return full_path
    except ValueError:
        raise ValueError(f"Path {user_path} is outside base directory")

# Usage:
base = Path.home() / "projects"
user_input = self.location_var.get()
try:
    safe_path = sanitize_path(base, user_input)
except ValueError as e:
    messagebox.showerror("Invalid Path", str(e))
    return
